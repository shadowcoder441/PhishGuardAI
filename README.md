# PhishGuardAI

AI-powered phishing detection and cyber threat analysis platform built using FastAPI, React, and Machine Learning.

---

## Features

- Phishing URL Detection
- Threat Confidence Scoring
- Machine Learning-based Analysis
- FastAPI Backend
- React Frontend
- REST API Integration
- Real-time Prediction System

---

## Tech Stack

### Frontend
- React
- Vite
- Axios
- Tailwind CSS

### Backend
- FastAPI
- Python

### Machine Learning
- Scikit-learn
- Logistic Regression
- TF-IDF Vectorization
